This archive documents every significant iteration of the RX-78 Gundam kit line. Data is stored in a flat kits.json file and loaded at runtime — no backend required. Hosted on GitHub Pages.

To add a kit: edit kits.json in the repository, commit, and push. Photos go in photos/{GRADE}/{kit-id}/ as front.jpg, back.jpg, and 01.jpg, 02.jpg etc.
